Tag <img>

Exemplo

Como inserir uma imagem:

<img src="img_girl.jpg" alt="Girl in a jacket" width="500" height="600">

Definição e Uso

A tag é usada para incorporar uma imagem em uma página HTML.
As imagens não são tecnicamente inseridas em uma página da web; as imagens são vinculadas a páginas da web. A tag cria um espaço de retenção para a imagem referenciada.
A tag tem dois atributos obrigatórios:
src - especifica o caminho para a imagem
alt - especifica um texto alternativo para a imagem, se a imagem por algum motivo não puder ser exibida
Nota: Além disso, sempre especifique a largura e a altura de uma imagem. Se largura e altura não forem especificadas, a página pode piscar enquanto a imagem carrega.
Dica: para vincular uma imagem a outro documento, simplesmente aninhe a tag <img> dentro de uma tag <a> (veja o exemplo abaixo).

Atributos

Atributo Valor Descrição
alt text Especifica um texto alternativo para uma imagem
crossorigin anonymous
use-credentials		 Permitir que imagens de sites de terceiros que permitem acesso de origem cruzada sejam usadas com o canvas
height pixels Especifica a altura de uma imagem
ismap ismap Especifica uma imagem como um mapa de imagem do lado do servidor
loading eager
lazy		 Especifica se um navegador deve carregar uma imagem imediatamente ou adiar o carregamento de imagens até que algumas condições sejam atendidas
longdesc URL Especifica um URL para uma descrição detalhada de uma imagem
referrerpolicy no-referrer
no-referrer-when-downgrade
origin
origin-when-cross-origin
unsafe-url		 Especifica quais informações de referência usar ao buscar uma imagem
sizes sizes Especifica tamanhos de imagem para diferentes layouts de página
src URL Especifica o caminho para a imagem
srcset URL-list Especifica uma lista de arquivos de imagem para usar em diferentes situações
usemap #mapname/td> Especifica uma imagem como um mapa de imagem do lado do cliente
width pixels Especifica a largura de uma imagem
