Tag <h1>to<h6>

Exemplo

Os seis cabeçalhos HTML diferentes:

<h1>This is heading 1</h1>
<h2>This is heading 2</h2>
<h3>This is heading 3</h3>
<h4>This is heading 4</h4>
<h5>This is heading 5</h5>
<h6>This is heading 6</h6>

Definição e Uso

As tags <h1> a <h6> são usadas para definir cabeçalhos HTML.
<h1> define o cabeçalho mais importante. <h6> define o título menos importante.
Observação: use apenas um <h1> por página - deve representar o título / assunto principal de toda a página. Além disso, não pule níveis de título - comece com <h1>, a seguir use <h2> e assim por diante.
