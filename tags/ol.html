Tag <ol>

Exemplo

Duas listas ordenadas diferentes (a primeira lista começa em 1 e a segunda começa em 50):

<ol>
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ol>

<ol start="50">
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ol>

Definição e Uso

A tag <ol> define uma lista ordenada. Uma lista ordenada pode ser numérica ou alfabética.
A tag <li> é usada para definir cada item da lista.
Dica: Use CSS para estilizar listas.
Dica: Para lista não ordenada, use a tag <ul>.

Atributos

Atributo Valor Descrição
reversed reversed Especifica que a ordem da lista deve ser invertida (9,8,7 ...)
start number Especifica o valor inicial de uma lista ordenada
