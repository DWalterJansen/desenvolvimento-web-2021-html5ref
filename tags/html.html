Tag <html>

Exemplo

Um documento HTML simples:

<!DOCTYPE html>
<html lang="en">
<head>
  <title>Title of the document</title>
</head>
<body>

<h1>This is a heading</h1>
<p>This is a paragraph.</p>

</body>
</html>

Definição e Uso

A tag <html> representa a raiz de um documento HTML.
A tag <html> é o contêiner para todos os outros elementos HTML (exceto para a tag <! DOCTYPE>).
Nota: Você deve sempre incluir o atributo lang dentro da tag <html>, para declarar o idioma da página web. Isso serve para auxiliar os mecanismos de pesquisa e navegadores.

Atributos

Atributo Valor Descrição
xmlns http://www.w3.org/1999/xhtml Especifica o atributo de namespace XML (se você precisa que seu conteúdo esteja em conformidade com XHTML)

Suporte de Navegador

Tag
<html>

Exemplo

A maioria dos navegadores exibirá o elemento <html> com os seguintes valores padrão:

html {
  display: block;
}

html:focus {
  outline: none;
}
