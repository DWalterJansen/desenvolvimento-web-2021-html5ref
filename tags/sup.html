Tag <sup>

Exemplo

Texto sobrescrito:

<p>This text contains <sup>superscript</sup> text.</p>

Definição e Uso

A tag <sup> define o texto sobrescrito. O texto sobrescrito aparece com meio caractere acima da linha normal e às vezes é renderizado em uma fonte menor. Texto sobrescrito pode ser usado para notas de rodapé, como WWW.
Dica: Use a tag <sup> para definir o texto subscrito.
