Definição e Uso

A tag <embed> define um contêiner para um recurso externo, como uma página da web, uma imagem, um reprodutor de mídia ou um aplicativo de plug-in.

Atributos

Atributo Valor Descrição
height pixels Especifica a altura do conteúdo incorporado
src URL Especifica o endereço do arquivo externo para incorporar
type media_type Especifica o tipo de mídia do conteúdo incorporado
width pixels Especifica a largura do conteúdo incorporado
